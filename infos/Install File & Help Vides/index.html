1- Install Theme & Option Tree

Step One: Upload 'city.zip' theme

Step Two: Activate City Theme

Step Three: Go to http://wp.envato.com and Download Option Tree Plugin

Step Four: Upload 'Option Tree' plugin .zip file

Step Five: Activate 'Option Tree' plugin

Step Six: Click 'Option Tree' button and, Drop Down Menu - Settings click

Step Seven: Click 'Import' area and, Select 'setup.xml' and upload.

Step Eight: Finish!

2- CHMODE Options (Tibthumb Thumbnail Script)

Solution 1:
If you not see thumb image, open your web ftp, theme folder / wp-content/themes/life/script/ right click 'cache' folder and change permission (chmode 655)

Solution 2:
If you not see thumb image, open your web ftp, theme folder / wp-content/themes/life/script/ right click 'cache' folder and change permission (chmode 777)

Solution 3:
If you not see thumb image, open your web ftp, theme folder / wp-content/themes/life/script/ right click 'cache' folder and change permission (chmode 777) and change wp-content folder chmode (777)
