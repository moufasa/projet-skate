== Installation ==

1. Upload `plugin directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add questions and answers
4. Place `[faq collect cat=3]` in your templates 


== Frequently Asked Questions ==
	
= How can i use HTML FAQ Page?

* After activation, HTML FAQ Page will create a new custom type - Questions -where
you can add questions and answers as much as you need.
* Insert this code <code>[faq_collect cat=3]</code> any where on your posts or pages.
* See you web site.

=  The plugin limits only up to 10 questions to display on a page. How to display all questions that I have created?

* To show more than 10 questions you need to go to Admin Area -> Settings -> Reading Settings
* Over there set the ' Blog pages show at most ' into a new value (20) for example ! 
 
 = How can i re-order the questions?

* Login into your dashboard=>Configuration:
* Questions are listed by categories
* To re-order your questions, use drag and drop :) 
* Done :)
